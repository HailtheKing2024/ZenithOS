# Zenith Theme

The first theme layer is a small set of CSS tokens and GNOME defaults. It is
not a full GTK theme fork. GTK/libadwaita should remain the baseline so apps
stay compatible and maintainable.

## Direction

- neutral dark and light surfaces
- green-blue Zenith accent
- 8px normal radius
- restrained shadows
- high contrast text
- no decorative gradients or blobs

## Files

- `zenith.css`: token reference for shell and app scaffolds
